# OtomeBall
用Unity实现的舞乙HIME的弹珠游戏，独立游戏。<br>
scenes下的Login为启动场景。<br>
在Unity还是4.6的时候编写，核心还是UGUI，大概为2014年吧。<br>
现在提交补个档。<br>

configData.cs和对应的config是通过工具由excel自动生成。<br>
由于编码不一样，在游戏内目前配置中文为乱码，后面再修复。<br>
